How hipster is your favorite artist?

Hipsterify is a small application that use Spotify API to search and rank artists according to their popularity on the platform. The popularity* of the artist is always between 0 and 100, with 100 being the most popular. But Hipsterify does the opposite, with 100 being the least popular and therefore the most hipster.

Hipsterify is an experiment by Ricardo Ramirez. This project is developed in Angular.js using angular-mass-autocomplete and angular-spotify. The source code is on GitHub and is still under development.

* The artist's popularity is calculated from the popularity of all the artist's tracks.
